# Tour Du Monde En 5 Pas

## Description
Ce site web a été développé dans le cadre d'un projet durant la première année de BUT Informatique. Ce site web permet de découvrir le monde en 5 pas avec des suggestions pour rendre ce voyage plus agréable.
<br>
<br>
Ce projet a été réalisé par un groupe de 4 personnes. En amont, un mood board, une maquette ainsi que des documents (cahier des charges, répartition des tâches et missions) ont été faits afin de connaître les besoins du client et de le tenir informé de l'état du projet.

## Exécution
Télécharger les documents et dossiers, puis lancer le fichier `index.html`.
